# 🧩 Full-Stack MERN Application

> A modern full-stack web application built with **React (Vite), Node.js, Express, and MongoDB**.

This project demonstrates scalable full-stack architecture with modern frontend tooling, robust backend design, and production-ready patterns such as validation, API handling, and state management.

---

## ✨ Features

### 📌 Core Functionality
- Full CRUD operations
- REST API integration
- Form validation
- Real-time UI updates
- Responsive design

---

### 🎨 Frontend (React + Vite)
- React 19 with Hooks
- Chakra UI component system
- React Router navigation
- React Query for server state
- React Hook Form + Zod validation
- Framer Motion animations
- Axios API layer
- Toast notifications (React Hot Toast)

---

### ⚙️ Backend (Node.js + Express)
- RESTful API architecture
- MongoDB + Mongoose ODM
- Environment configuration (dotenv)
- Modular backend structure
- Error handling middleware
- Scalable controller/service design

---

### 🗄️ Database (MongoDB)
- Schema-based modeling (Mongoose)
- Flexible document structure
- Scalable NoSQL design

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
## Backend Setup
```bash
cd backend
npm install
```
```bash
PORT=5000
MONGO_URI=your_mongodb_connection_string
```

### FRONTEND SETUP
```bash
cd frontend
npm install
```
### 1. Create frontend/.env
```bash
VITE_API_URL=http://localhost:5000/api
```

### 2. Run the Application
Terminal 2 - Frontend Development
```bash
cd backend
npm run dev
```
Server runs at: http://localhost:5000

Terminal 2 - Frontend Development
```bash
cd frontend
npm run dev
```
App runs at: http://localhost:5173
